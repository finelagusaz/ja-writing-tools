# 出力サンプル

各モードでの校正結果の出力例。

---

## サンプル入力

以下のテキストを校正する場合を例とする。

```
弊社のサービスについてご説明させて頂きます。

このシステムは、ユーザが簡単に設定を見れる様に設計されています。以外と知られていませんが、
データーベースへのアクセスも可能です。

まず最初に、基本的な使い方を説明します。設定画面を開いて頂き、必要な項目を入力して下さい。
入力が完了したら、保存ボタンを押して頂ければと思います。

ご不明な点がございましたら、お気軽にお問い合わせ下さい。担当者が対応させて頂きます。
```

## 標準モードの出力

```markdown
## 校正結果

### 修正提案（確信度：高）

| 箇所 | 現在の表記 | 修正案 | 理由 |
|---|---|---|---|
| 1段落 | させて頂きます | させていただきます | 補助動詞はひらがな |
| 2段落 | 見れる | 見られる | ら抜き言葉 |
| 2段落 | 様に | ように | 形式名詞はひらがな |
| 2段落 | 以外と | 意外と | 同音異義語（「思いがけず」の意味） |
| 2段落 | データーベース | データベース | 「ー」不要（database） |
| 3段落 | まず最初に | まず / 最初に | 重複表現 |
| 3段落 | 開いて頂き | 開いていただき | 補助動詞はひらがな |
| 3段落 | 下さい | ください | 補助動詞はひらがな |
| 4段落 | 下さい | ください | 同上 |
| 4段落 | させて頂きます | させていただきます | 同上 |

### 表記揺れ（確信度：中）

| 表記A | 出現 | 表記B | 出現 | 推奨 |
|---|---|---|---|---|
| ユーザ | 1箇所 | （他になし） | - | 「ユーザー」に統一を検討 |

### 要確認（確信度：低）

| 箇所 | 表記 | 確認事項 |

| 全体 | 〜させていただきます（2回） | 頻度は許容範囲だが、1回は「いたします」に置き換えも可 |

ご不明な点がございましたら、お気軽にお問い合わせください。担当者が対応させていただきます。
```

## 簡易モードの出力

```markdown
## 校正結果（簡易チェック）

以下の明らかな誤りを検出しました：

- 「以外と」→「意外と」（同音異義語）
- 「見れる」→「見られる」（ら抜き言葉）
- 「データーベース」→「データベース」
- 「まず最初に」→「まず」または「最初に」（重複）
- 「頂き」「下さい」→「いただき」「ください」（補助動詞はひらがな、計5箇所）

修正版が必要な場合はお知らせください。
```

## 詳細モードの出力

標準モードの内容に加えて、以下を含む：

```markdown

- 文体：敬体（です・ます）で統一されています
- 敬語レベル：丁寧（「させていただく」の使用あり）
- 読点の傾向：適切

### スタイル改善の提案（任意）

| 箇所 | 現在の表記 | 改善案 | 理由 |

| 3段落 | 押して頂ければと思います | 押してください | 簡潔さを優先する場合 |
| 4段落 | 対応させていただきます | 対応いたします | 「させていただく」の頻度を下げる |

### 統計

- 総文字数：約280字
- 修正提案：10件（確信度：高）
- 表記揺れ：1件
- 要確認：2件
```

## 小説の場合の出力例

### 入力

```
「明日、会えれる？」
彼女はそう言って、僕の方を見た。
「うん、会えるよ」
僕は頷いた。全然問題ない。むしろ嬉しいくらいだ。
```

### 出力（標準モード）

```markdown
## 校正結果

### 修正提案（確信度：高）

（なし）

### 要確認（確信度：低）

| 箇所 | 表記 | 確認事項 |
|---|---|---|
| 1行目 | 会えれる | れ足す言葉。会話文のため口語表現として意図的であれば問題なし |
| 4行目 | 全然問題ない | 「全然」+肯定。現代口語では許容されるが、地の文のため確認 |

### コメント

会話文と地の文で文体が適切に使い分けられています。
「会えれる」は文法的には「会える」が正しいですが、キャラクターの口調として意図的であれば修正不要です。

---

## 修正版

（修正なし。原文のまま）
```
