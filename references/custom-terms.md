# ユーザー固有の用語集

このファイルを編集して、プロジェクト固有の用語や表記ルールを登録してください。
校正時にこのファイルが参照され、登録された表記が優先されます。

## 使い方

1. 下記のテンプレートに従って用語を追加
2. 校正実行時に自動的に参照される
3. 標準ルールより優先される

## 許容する表現

通常は誤りとされるが、このプロジェクトでは意図的に使用する表現。

| 表現 | 通常の指摘 | 許容理由 |
|---|---|---|
| 全然良い | 「全然」は否定と呼応 | 口語的な文体のため |
<!-- 追加例:
| 見れる | ら抜き言葉 | 会話文で使用 |
| 〜とか〜とか | 書き言葉では避ける | カジュアルなブログ記事 |
-->

## 禁止表現

このプロジェクトで使用を避けたい表現。

| 禁止表現 | 代替表現 | 理由 |
|---|---|---|
| させていただく | いたします | 過剰敬語を避ける |
<!-- 追加例:
| 〜になります | 〜です | 冗長表現を避ける |
| 弊社 | 当社 | 社内スタイルガイド |
-->

## 文体設定

以下のコメントを外して設定を有効化してください。

```yaml
# 文体: です・ます / だ・である / 混在可
style: です・ます

# 一人称: 私 / 僕 / 俺 / わたし / 統一しない
# first_person: 私

# 長音符: あり / なし / 統一しない
long_vowel: あり

# 数字: 半角 / 全角 / 統一しない
numbers: 半角

# 英字: 半角 / 全角 / 統一しない
alphabet: 半角

# 句読点: 。、 / ．， / 統一しない
# punctuation: 。、
```

---

## 設定例

### 技術ブログ向け

```yaml
style: です・ます
long_vowel: なし
numbers: 半角
alphabet: 半角
```

許容表現：
- 「全然」+ 肯定（カジュアルな文体）

### 小説向け

```yaml
style: だ・である
first_person: 僕
numbers: 全角
```

許容表現：
- 会話文中のら抜き言葉
- キャラクター固有の口調
