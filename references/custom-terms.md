# ユーザー固有の用語集

このファイルを編集して、プロジェクト固有の用語や表記ルールを登録してください。
校正時にこのファイルが参照され、登録された表記が優先されます。

## 使い方

1. 下記のテンプレートに従って用語を追加
2. 校正実行時に自動的に参照される
3. 標準ルールより優先される

## 文体設定

```
# 以下のコメントを外して設定を有効化

# 文体: です・ます / だ・である / 混在可
# style: です・ます

# 一人称: 私 / 僕 / 俺 / わたし / 統一しない
# first_person: 私

# 長音符: あり / なし / 統一しない
# long_vowel: あり

# 数字: 半角 / 全角 / 統一しない
# numbers: 半角

# 英字: 半角 / 全角 / 統一しない
# alphabet: 半角
```
