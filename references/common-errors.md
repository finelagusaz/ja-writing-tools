## 助詞・形式名詞の誤り

### ひらがな表記が正しい助詞

| 誤 | 正 | 解説 |
|---|---|---|
| 〜に於いて | 〜において | ひらがな推奨 |
| 〜に取って | 〜にとって | 形式助詞はひらがな |
| 〜に因って | 〜によって | 形式助詞はひらがな |
| 〜と言う | 〜という | 形式名詞はひらがな（「〜と言う」が動作の場合は漢字可） |
| 〜と言った | 〜といった | 形式名詞はひらがな |
| 〜と共に | 〜とともに | ひらがな推奨 |
| 〜に付いて | 〜について | 形式助詞はひらがな（「壁に付いて」など物理的付着は漢字） |
| 何故 | なぜ | ひらがな推奨 |
| 其処 | そこ | ひらがな推奨 |
| 〜する事 | 〜すること | 形式名詞はひらがな（「大事な事」など実質名詞は漢字可） |
| 〜した時 | 〜したとき | 形式名詞はひらがな（「その時」など時点を指す場合は漢字可） |
| 〜する様に | 〜するように | 形式名詞はひらがな（「その様に」など様態は漢字可） |
| 〜する為に | 〜するために | 形式名詞はひらがな |

| 誤 | 正 | 解説 |

| 〜をして | 〜として | 「として」が正 |
| 〜のとして | 〜として | 「の」不要 |
| 〜とゆう | 〜という | 「いう」が正 |
| 〜たり〜たり | 〜たり〜たりする | 「する」で結ぶ |

| 誤 | 正 | 意味 |

| 役不足 | 力不足 | 「役不足」=能力に対して役が軽い |
| 気が置けない | 気が置ける | 「気が置けない」=遠慮がいらない |
| 煮詰まる | 行き詰まる | 「煮詰まる」=結論に近づく（良い意味） |
| 姑息 | 卑怯 | 「姑息」=一時しのぎ |
| 檄を飛ばす | 叱咤激励する | 「檄」は自分の主張を広める文書 |
| 爆笑 | 大笑い | 「爆笑」=大勢で笑う |
| 失笑 | 苦笑 | 「失笑」=こらえきれず笑う |
| 憮然 | 不満そう | 「憮然」=失望してぼんやり |
| 琴線に触れる | 逆鱗に触れる | 「琴線」=心の奥深い感動 |
| 汚名挽回 | 汚名返上 | 「名誉挽回」との混同 |

## 句読点の誤り

| 誤 | 正 | 解説 |
|---|---|---|
| 、、 | 、 | 二重は誤り |

## 間違いやすいが正しい表記

これらは誤りとして指摘されやすいが、実際には正しい表記です。

| 正しい表記 | よくある誤解 | 解説 |
|---|---|---|
| おかしかった | 形容詞の活用が誤りでは？ | 形容詞の過去形として正しい |
| 楽しかった | 同上 | 同上 |
| 全然良い（口語） | 全然は否定のみ？ | 現代口語では肯定も可（文章では避ける） |
| やむを得ない | 「已むを得ない」では？ | 「止む」が正しい |

## 技術用語・記号の表記選択

### IT用語の長音符表記

IT業界では長音符を省略する慣習がありますが、一般文書では長音符ありも許容されます。
**重要なのは文書内で統一すること**です。

| 長音なし（IT業界標準） | 長音あり（一般） |
|---|---|
| サーバ | サーバー |
| ユーザ | ユーザー |
| コンピュータ | コンピューター |
| ディレクトリ | ディレクトリー |
| パラメータ | パラメーター |

### 長音符を残すのが一般的な語

以下の語は長音符を省略しないのが一般的です。

| 標準表記 | 理由 |
|---|---|
| エラー | 2音節で短いため |
| データベース | 複合語の後半 |
| フォーマット | 一般化した外来語 |
| アルゴリズム | 学術用語として定着 |

### カタカナ表記の揺れ（要統一）

| 表記A | 表記B | 判断基準 |
|---|---|---|
| キャッシュ | キャシュ | 英語発音準拠なら「キャッシュ」 |
| ディレクトリ | ディレクトリー | 業界・文書スタイルで統一 |
| デフォルト | ディフォルト | 「デフォルト」が一般的 |
| シミュレーション | シュミレーション | 「シミュレーション」が正（simulationの発音） |
| コミット | コミット | 促音「ッ」の有無を統一 |

### 記号の表記統一

| 記号A | 記号B | 判断基準 |
|---|---|---|
| 〜（全角波ダッシュ） | ～（全角チルダ） | 文書内で統一（JIS規格では波ダッシュ「〜」推奨） |
| ー（長音符） | 〜（波ダッシュ） | 「サーバー」は長音符、「10〜20」は波ダッシュ |
| 0123（全角数字） | 0123（半角数字） | 文書内で統一（技術文書では半角推奨） |
| ＡＢＣ（全角英字） | ABC（半角英字） | 半角推奨（全角は特殊な強調以外避ける） |
| ()（半角括弧） | （）（全角括弧） | 日本語文中では全角、英数のみなら半角 |
| 【】（隅付き括弧） | 「」（鉤括弧） | 見出し・強調は【】、引用・発言は「」 |
