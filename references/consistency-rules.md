# 表記統一ルール

## 漢字/ひらがな選択

### 基本方針

- 小説: 柔らかい印象を優先 → ひらがな多め
- 技術文書: 正確さを優先 → 漢字多め

### 判断が分かれる語

| 語 | 小説推奨 | 技術文書推奨 | 備考 |
|---|---|---|---|
| 事/こと | こと | 事 | 名詞化は「こと」優先 |
| 時/とき | とき | 時 | |
| 物/もの | もの | 物 | 抽象概念は「もの」 |
| 等/など | など | 等 | |
| 所/ところ | ところ | 所 | 形式名詞は「ところ」 |
| 訳/わけ | わけ | わけ | 両方ひらがな推奨 |
| 様/よう | よう | よう | 形式名詞は「よう」 |
| 為/ため | ため | ため | 両方ひらがな推奨 |
| 程/ほど | ほど | 程 | 程度の意味は「ほど」 |
| 通り/とおり | とおり | 通り | 形式名詞は「とおり」 |
| 故/ゆえ | ゆえ | 故 | 文語的な文書のみ漢字 |
| 毎/ごと | ごと | 毎 | 「〜するたびに」は「ごと」 |

### 動詞の漢字/ひらがな

| 漢字表記 | ひらがな表記 | 使い分け |
|---|---|---|
| 言う | いう | 「〜という」は形式動詞なので「いう」 |
| 有る/在る | ある | 補助動詞は「ある」（〜してある） |
| 居る | いる | 補助動詞は「いる」（〜している） |
| 見る | みる | 補助動詞は「みる」（〜してみる） |
| 行く | いく | 補助動詞は「いく」（〜していく） |
| 来る | くる | 補助動詞は「くる」（〜してくる） |
| 貰う | もらう | 補助動詞は「もらう」（〜してもらう） |
| 遣る | やる | 「やる」が一般的 |
| 出来る | できる | 可能の意味は「できる」 |

## カタカナ長音符

### 確認項目

- 全体で統一されているか
- 業界標準に従っているか

### 主な語例

| 長音あり | 長音なし | 備考 |
|---|---|---|
| コンピューター | コンピュータ | IT業界は長音なし多い |
| サーバー | サーバ | 同上 |
| ユーザー | ユーザ | 同上 |
| データベース | データベース | 「ス」は長音扱いしない |
| プリンター | プリンタ | |
| スキャナー | スキャナ | |
| ブラウザー | ブラウザ | |
| フォルダー | フォルダ | |
| パラメーター | パラメータ | |
| プロセッサー | プロセッサ | |

### 長音符を省略しないのが一般的な語

| 語 | 理由 |
|---|---|
| エラー | 2音節で短いため |
| データ | 定着した表記 |
| エリア | 定着した表記 |
| アイデア | 定着した表記 |

## 英数字の全角/半角

### 数字

| 種類 | 推奨 | 例 |
|---|---|---|
| 技術文書の数値 | 半角 | 100個、2024年 |
| 小説の数値 | 全角または漢数字 | １００個、百個 |
| 桁区切り | カンマ（半角） | 1,000,000 |
| 小数点 | ピリオド（半角） | 3.14 |
| 日付 | 半角 | 2024/01/01, 2024-01-01 |
| 電話番号 | 半角ハイフン | 03-1234-5678 |

### アルファベット

| 種類 | 推奨 | 例 |
|---|---|---|
| 英単語 | 半角 | API, JavaScript |
| 略語 | 半角 | AI, IT, OS |
| 固有名詞 | 半角 | Google, Apple |
| 強調目的 | 全角も可 | ＡＢＣＤ（稀） |

### 記号

| 記号 | 日本語文中 | 英数字のみ |
|---|---|---|
| 括弧 | 全角（） | 半角() |
| コロン | 全角： | 半角: |
| セミコロン | 全角； | 半角; |
| クエスチョン | 全角？ | 半角? |
| エクスクラメーション | 全角！ | 半角! |
| スラッシュ | 全角／ | 半角/ |
| ハイフン | 全角ー（長音）| 半角- |

## 句読点

### 種類の選択

| 種類 | 用途 | 例 |
|---|---|---|
| 。、（テン・マル） | 一般的な日本語文書 | 私は、学生です。 |
| ．，（ピリオド・カンマ） | 学術論文、公文書 | 私は，学生です． |
| 。，（混合） | 避ける | - |

### 句読点の打ち方

| ルール | 良い例 | 悪い例 |
|---|---|---|
| 主語の後（長い場合） | 昨日駅で会った友人は、元気だった。 | 昨日駅で会った友人は元気だった。 |
| 接続詞の後 | しかし、彼は来なかった。 | しかし彼は来なかった。 |
| 並列要素の区切り | りんご、みかん、バナナ | りんごみかんバナナ |
| 文節ごとには不要 | 私は昨日、駅で友人に会った。 | 私は、昨日、駅で、友人に、会った。 |

## スペースの扱い

### 全角/半角スペース

| 場面 | ルール | 例 |
|---|---|---|
| 日本語と英数字の間 | スペースなし or 半角 | 統一する |
| 英単語間 | 半角スペース | Hello World |
| 日本語文中 | スペースなし | 今日は晴れです |
| 箇条書きの先頭 | 統一する | ・項目1 |

### 推奨パターン

```
# 日本語と英数字の間のスペース

## パターンA: スペースなし（推奨）
これはAPIです。
version2.0を使用。

## パターンB: 半角スペースあり
これは API です。
version 2.0 を使用。

※どちらでも可。文書内で統一すること。
```

## 括弧の使い分け

| 括弧 | 用途 | 例 |
|---|---|---|
| 「」 | 引用、発言、強調 | 彼は「行く」と言った。 |
| 『』 | 書名、作品名、「」内の引用 | 『源氏物語』を読んだ。 |
| （） | 補足説明、読み仮名 | 東京（とうきょう） |
| 【】 | 見出し、ラベル、強調 | 【重要】お知らせ |
| 〈〉 | 引用内の引用、やや弱い強調 | 〈注意〉してください |
| 《》 | 二重山括弧、作品名 | 《モナリザ》 |
| ［］ | 注釈、編集者追記 | ［原文ママ］ |

### 括弧と句読点の位置

| パターン | 例 | 備考 |
|---|---|---|
| 句点は括弧の外 | 〜です（補足）。 | 一般的 |
| 括弧内が独立文 | （これは補足です。） | 括弧内にも句点 |
| 発言の終わり | 「行きます」 | 括弧内に句点なし |
| 発言が文末 | 彼は「行く」と言った。 | 括弧外に句点 |

## 文体の一貫性

### 敬体/常体

| 文体 | 特徴 | 用途 |
|---|---|---|
| 敬体（です・ます） | 丁寧、読者との距離感 | ブログ、マニュアル、ビジネス文書 |
| 常体（だ・である） | 簡潔、断定的 | 論文、小説、報告書 |

### 混在のチェック

```
# 敬体→常体の混在（要修正）
これは重要な機能です。処理が高速だ。

# 修正例
これは重要な機能です。処理が高速です。
```

### 許容される混在

| パターン | 例 | 備考 |
|---|---|---|
| 引用内 | 「これは問題だ」と彼は言いました。 | 引用は元の文体を維持 |
| 箇条書き | ・処理が高速である / 本文は「です・ます」 | 箇条書きは常体も可 |
| 見出し | 【注意事項】/ 本文は「です・ます」 | 見出しは常体も可 |

## 一人称の統一

| 一人称 | 印象 | 用途 |
|---|---|---|
| 私（わたし） | フォーマル、中性 | ビジネス、論文 |
| 私（わたくし） | より丁寧 | 式典、公式文書 |
| 僕 | カジュアル、男性的 | ブログ、エッセイ |
| 俺 | くだけた、男性的 | 小説の一人称 |
| 自分 | 中性、やや硬い | 報告書 |
| 筆者 | 客観的 | 論文、解説 |
| 著者 | 客観的 | 書籍 |

## 接続詞の統一

### 逆接

| 表現 | フォーマル度 | 備考 |
|---|---|---|
| しかし | 高 | 論文、ビジネス |
| だが | 中 | 常体の文章 |
| けれども | 中 | 敬体の文章 |
| でも | 低 | 口語的 |
| ただ | 中 | 補足的な逆接 |

### 順接

| 表現 | フォーマル度 | 備考 |
|---|---|---|
| したがって | 高 | 論理的帰結 |
| そのため | 高 | 因果関係 |
| だから | 中 | 常体向き |
| ですから | 中 | 敬体向き |
| なので | 低 | 文頭では避ける |

### 添加

| 表現 | フォーマル度 | 備考 |
|---|---|---|
| また | 高 | 汎用的 |
| さらに | 高 | 追加・強調 |
| 加えて | 高 | フォーマル |
| そして | 中 | 時系列にも |
| それに | 低 | 口語的 |
